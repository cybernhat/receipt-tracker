What is convex?
 - convex is a modern backend framework written with Typescript.
 - it handles:
    Database
    APIs
    real-time updates
    server logic
 - essentially database + api in one as opposed to database seperated from api
 
 - query: access db
 - mutation: changes db
 - action: used for external api
